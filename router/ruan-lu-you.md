## X86平台

只要是台电脑就行，不限于Intel / AMD

**可以用家里的旧电脑改装，0成本**  
也可以到TB/闲鱼淘二手矿渣小主机，基本在200-600之间  
要么直接TB买成品软路由，600起价，配置高的例如搭载I7 CPU的成品软路由可以打上2500  
或者捡垃圾收拆机件自己组电脑，组全新电脑也没有问题  
你拿十年前的赛扬也能用，装一台3990X也可以，全看个人需求

{% hint style="warning" %}
此坑深不见底，前期投入不一定大，后期需求增长导致的开销增长极高，慎入
{% endhint %}
